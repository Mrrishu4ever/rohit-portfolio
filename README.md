## ✨ About the Project

A modern, animated personal portfolio website built to showcase skills, projects, and experience as a Full Stack Web Developer. Designed with a focus on smooth animations, clean UI, and live project demos.

- ⚡ **Animated Hero Section** — Smooth entrance animations

- 📱 **Fully Responsive** — Mobile, tablet & desktop

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Mrrishu4ever/rohit-portfolio.git

# Go to project folder
cd rohit-portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser!
